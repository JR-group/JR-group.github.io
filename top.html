JRgroup Projects

ABCD (Accurate Booleanization of Continuous Dynamics)

ABCD is a suite of machine-learning techniques for capturing dynamical system behaviour as discrete models.

  • ABCD homepage: being set up

PHLOGON (PHase LOGic using Oscillatory Nanosystems)

PHLOGON is a scheme that uses oscillators to implement Ising and von Neumann machines.

  • PHLOGON homepage: http://draco.eecs.berkeley.edu/PHLOGON/

MAPP (The Berkeley Model and Algorithm Prototyping Platform)

Berkeley MAPP is a MATLAB-based platform for prototyping numerical models and simulation algorithms.

  • MAPP homepage: http://mapp.eecs.berkeley.edu
